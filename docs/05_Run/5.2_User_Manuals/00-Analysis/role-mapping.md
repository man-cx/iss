# SAP S/4HANA End-User Role Mapping
## Home Control Company

---

## Purpose
This document maps business processes to specific end-user roles by department, providing a clear view of who performs what activities in the SAP system.

---

## 1. FINANCE DEPARTMENT ROLES

### 1.1 GL Processing Officer
**Primary Responsibilities:**
- Post general ledger journal entries
- Process manual GL adjustments
- Reverse GL postings when errors are found
- Maintain supporting documentation

**Key Processes:**
- Manage Posting in GL and Parallel Ledger

**Key Applications:**
- Manage Journal Entries
- Display G/L Account Balances
- Display Line Items in General Ledger

---

### 1.2 GL Administrator (Local)
**Primary Responsibilities:**
- Extend GL accounts to company code level
- Fill GL maintenance request forms
- Verify GL account requirements
- Communicate with central GL administration

**Key Processes:**
- Manage GL Master Records

**Key Applications:**
- Manage G/L Account Master Data

---

### 1.3 GL Administration - Central
**Primary Responsibilities:**
- Create GL accounts at Chart of Accounts level
- Approve GL account creation requests
- Assign GL accounts to Financial Statement version
- Maintain global GL structure

**Key Processes:**
- Manage GL Master Records

**Key Applications:**
- Manage G/L Account Master Data
- Define Financial Statement Versions

---

### 1.4 GL Executive Periodic
**Primary Responsibilities:**
- List and analyze GL parked documents
- Perform GR/IR Auto Clearing
- Open and close MM/FI posting periods
- Execute GL periodic postings
- Process AR/AP/GL adjustments
- Perform open item clearing
- Execute foreign currency revaluation
- Run month-end closing activities
- Execute balance carryforward

**Key Processes:**
- Month End Closing Operations
- Manage Balance Carryforward

**Key Applications:**
- Schedule General Ledger Jobs
- Close Periods
- Manage Posting Periods
- Clear Open Items
- Automatic Clearing
- Manage Journal Entries
- Manage Recurring Journal Entries
- Display G/L Account Balances

---

### 1.5 GL Reporting
**Primary Responsibilities:**
- Run financial reports
- Analyze financial data
- Generate balance sheets and income statements

**Key Processes:**
- Month End Closing Operations

**Key Applications:**
- Balance Sheet / Income Statement China
- Display G/L Account Balances
- Display Line Items in General Ledger

---

### 1.6 Finance Manager (GL Reporting)
