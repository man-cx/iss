## SECTION 2: GETTING STARTED

### 2.1 Logging into SAP

**Step-by-Step Instructions:**

1. Open your web browser (Chrome or Edge recommended)
2. Navigate to the SAP login page: [URL]
3. Enter your User ID
4. Enter your Password
5. Click "Log On"

**📸 Screenshot Placeholder:** SAP Login Screen

**✏️ Note:** Your User ID and initial password will be provided by IT. You will be prompted to change your password on first login.

**⚠️ Important:** Never share your SAP password with anyone.

### 2.2 Navigating the SAP Interface

**SAP Fiori Launchpad:**

The SAP Fiori Launchpad is your home screen in SAP. It contains tiles for all the applications you can access.

**Key Areas:**
- **Top Bar:** Contains user settings, notifications, and help
- **Tile Area:** Shows all available applications organized by groups
- **Search Bar:** Quickly find applications by name

**📸 Screenshot Placeholder:** SAP Fiori Launchpad

### 2.3 Finding Applications

**Method 1: Using Tiles**
1. Locate the application tile on your home screen
2. Click the tile to open the application

**Method 2: Using Search**
1. Click the search icon in the top bar
2. Type the name of the application
3. Click the application from the results

**Method 3: Using Menu**
1. Click the menu icon (☰) in the top left
2. Browse through the application groups
3. Click the application name

### 2.4 Basic Functions

**Common Buttons:**
- **Save** 💾 - Saves your work
- **Cancel** ✖ - Cancels and closes without saving
- **Check** ✓ - Validates your entries for errors
- **Copy** 📋 - Creates a copy of the current document
- **Display** 👁 - Opens in view-only mode
- **Change** ✏️ - Opens in edit mode

**Keyboard Shortcuts:**
- `Ctrl + S` - Save
- `Ctrl + F` - Find
- `F3` - Back
- `F8` - Execute/Search
